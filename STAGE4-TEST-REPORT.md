## 🔧 修复的Bug

### Bug #1: macOS脚本语法错误

**文件**: `scripts/install-macos.sh`
**位置**: 第29行
**错误**: `if [[ $(uname) != "Darwin" ]]; ]]; then` (多余的 `]]`)

**修复前**:
```bash
if [[ $(uname) != "Darwin" ]]; ]]; then
```

**修复后**:
```bash
if [[ $(uname) != "Darwin" ]]; then
```

---

### Bug #2: Docker脚本引用符错误

**文件**: `scripts/install-docker.sh`
**位置**: 第207行
**错误**: 使用了 `'` 而非 `"` 来闭合字符串

**修复前**:
```bash
-v $(pwd)/data:/home/openclaw/.openclaw/data \
'            openclaw/openclaw:latest
```

**修复后**:
```bash
-v $(pwd)/data:/home/openclaw/.openclaw/data \
            openclaw/openclaw:latest
```
