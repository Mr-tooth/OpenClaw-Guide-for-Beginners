# OpenClaw 2026.5.4~5.7 版本变更说明
发布时间：2026-05-14
适配教程版本：v1.7.1
---
## 📋 版本更新概要
v2026.5.4~v2026.5.7属于小版本迭代，**100%向下兼容v2026.4.x所有配置**，无破坏性变更，主要新增2个重要功能，修复多项稳定性问题。
---
## ✨ 新功能介绍
### 1. 内置文件传输插件
✅ 新增官方内置`file-transfer`插件，提供以下工具能力：
| 命令 | 功能说明 | 权限要求 |
|------|---------|---------|
| `file_fetch <文件路径>` | 读取指定路径文件内容 | 读权限 |
| `dir_list <目录路径>` | 列出目录下的文件和子目录 | 读权限 |
| `dir_fetch <目录路径>` | 批量获取目录下所有文件内容 | 读权限 |
| `file_write <文件路径> <内容>` | 写入内容到指定文件 | 写权限 |
> 插件默认已内置，无需额外安装，可直接通过工具调用使用。
### 2. Google Meet语音通话集成
✅ 新增Google Meet语音通话支持，通过Twilio拨号接入，支持：
- 实时语音通话交互
- Gemini语音桥接，语音转文字/文字转语音
- 通话中工具调用能力
-  paced音频流，播放更流畅

## 🔗 兼容性说明
✅ **完全向下兼容**：所有v2026.4.x版本的配置无需任何修改即可直接升级到v2026.5.7，无功能失效风险。

## 🚀 升级步骤
### 常规升级
```bash
# 1. 停止gateway服务
openclaw gateway stop
# 2. 升级OpenClaw
openclaw update
# 3. 自动修复配置（可选，推荐执行）
openclaw doctor --fix
# 4. 启动gateway服务
openclaw gateway start
# 5. 验证版本
openclaw version # 输出应为v2026.5.7
```
### Docker部署升级
```bash
# 拉取最新镜像
docker pull openclaw/clawdock:latest
# 重启容器
docker restart openclaw
```
