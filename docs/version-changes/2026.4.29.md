# OpenClaw 2026.4.29 版本变更说明
发布时间：2026-04-30

## 重大变更

### NVIDIA 模型支持
- ✅ 新增 NVIDIA 提供商支持
- ✅ 支持 API Key 入门引导
- ✅ 支持设置文档
- ✅ 支持静态目录元数据
- ✅ 支持字面量模型引用选择器支持

### 记忆系统升级
- ✅ 新增 People-aware Wiki 功能
- ✅ 支持来源视图
- ✅ 支持每会话 Active Memory 过滤器
- ✅ 支持超时部分召回
- ✅ 支持有界 REM 预览诊断

### 安全强化
- ✅ 新增 OpenGrep 扫描功能
- ✅ 更严格的 GHSA 分类策略
- ✅ 更安全的 exec/pairing/owner-scope 处理
- ✅ Docker/入门引导自动化
- ✅ web-fetch IPv6 ULA 选项

### Gateway 稳定性
- ✅ 支持慢主机启动
- ✅ 支持可重用模型目录
- ✅ 支持事件循环就绪诊断
- ✅ 支持运行时依赖修复
- ✅ 支持过期会话恢复
- ✅ 支持版本范围更新缓存

## 功能更新

### 消息和自动化
- ✅ Active-run 转向默认启用
- ✅ 可见回复强制执行
- ✅ 生成的子代理路由元数据
- ✅ 心跳提醒的可选后续承诺

### 提供商/模型覆盖
- ✅ NVIDIA 入门引导/目录

- ✅ 更安全的 Codex/OpenAI 兼容重放/流式传输

### 通道修复
- ✅ Slack Block Kit 限制
- ✅ Telegram 代理/Webhook/轮询/发送弹性
- ✅ Discord 启动/速率限制处理

- ✅ Microsoft Teams/Matrix/Feishu 边缘情况

## 安全修复

### 工具安全
- ✅ 配置的工具部分不再隐式扩展限制性配置文件
- ✅ 用户必须添加显式 `alsoAllow` 条目
- ✅ 更安全的 exec/pairing/owner-scope 处理

### 网络安全
- ✅ 新增运营商管理的出站代理路由
- ✅ 支持 `proxy.enabled` + `proxy.proxyUrl` 配置
- ✅ web-fetch IPv6 ULA 选项

### 插件安全
- ✅ 更安全的插件加载和执行
- ✅ 插件隔离改进

## 升级建议

### 推荐升级
- ✅ 强烈建议所有用户升级到此版本
- ✅ 特别推荐需要 NVIDIA 模型的用户升级
- ✅ 特别推荐需要高级记忆功能的用户升级
- ✅ 特别推荐需要增强安全性的用户升级

### 升级前准备
1. 备份当前配置文件
2. 备份自定义技能和插件
3. 记录当前提供商配置（如有）

### 升级步骤
```bash
# 1. 更新 OpenClaw
openclaw update

# 2. 自动修复配置
openclaw doctor --fix

# 3. 验证 NVIDIA 配置
openclaw config get nvidia

# 4. 验证记忆系统
openclaw memory status

# 5. 运行安全扫描
openclaw security scan
```
