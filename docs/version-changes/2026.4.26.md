# OpenClaw 2026.4.26 版本变更说明
发布时间：2026-04-28

## 重大变更

### Control UI/Talk 工作流
- ✅ 新增通用浏览器实时传输协议
- ✅ 新增 Google Live 浏览器 Talk 会话
- ✅ 新增 Gateway 中继，支持后端实时语音插件
- ✅ 新增 Control UI 原始配置差异面板
- ✅ 支持 JSON5 解析和敏感值脱敏

### Google Meet/Live 浏览器支持
- ✅ 新增日历支持的参会记录导出
- ✅ 新增导出清单功能
- ✅ 新增会议记录工具对等
- ✅ 新增 iframe 感知角色快照

### Cerebras 提供商集成
- ✅ 新增 Cerebras 作为内置插件
- ✅ 支持开箱即用的入门引导
- ✅ 支持静态模型目录

### 迁移工具
- ✅ 新增内置 Claude 导入器
- ✅ 新增 `openclaw migrate` 命令
- ✅ 支持计划、干运行和备份

## 功能更新

### CLI/模型改进
- ✅ 提供商过滤的模型列表通过显式源计划路由
- ✅ 支持稳定的权限顺序

### 插件/提供商处理
- ✅ 预运行时模型 ID 规范化移入插件清单
- ✅ 提供商端点元数据移入插件清单

### Control UI 增强
- ✅ 新增原始配置待处理变更差异面板
- ✅ 支持 JSON5 解析
- ✅ 支持敏感值脱敏

### Matrix E2EE 加密
- ✅ 新增 `openclaw matrix encryption setup` 命令
- ✅ 支持加密、恢复和验证

## 安全修复

### Gateway/设备令牌
- ✅ 停止从共享/管理响应中回显旋转的承载令牌

### Agent/会话生成
- ✅ 解析配置的裸模型别名用于生成模型覆盖

### Control UI/Talk
- ✅ 保持 Google Live 浏览器会话在 WebSocket 传输上
- ✅ 验证端点

### Gateway/启动
- ✅ 重用配置快照插件清单用于启动自动启用

## 性能优化

### 插件加载
- ✅ 使用缓存的发现模式快照加载进行只读插件能力查找
- ✅ 保持快照缓存与活动 Gateway 注册表隔离
- ✅ 使同插件通道/HTTP 路由重新注册幂等

### 模型发现
- ✅ 提供商过滤的模型列表通过显式源计划路由
- ✅ 支持稳定的权限顺序

## 命令变更

### 新增命令
- `openclaw migrate` - 迁移工具
- `openclaw matrix encryption setup` - Matrix E2EE 设置

### 配置变更
- 新增 Control UI 配置选项
- 新增 Cerebras 提供商配置
- 新增 Matrix E2EE 配置

## 兼容性说明

### 向后兼容
- ✅ 完全向下兼容 2026.4.x 版本配置
- ✅ Control UI 默认禁用，不影响现有用户
- ✅ 迁移工具可选，不影响现有工作流

### 迁移指南
- Control UI 配置自动初始化
- Cerebras 提供商自动发现
- Matrix E2EE 需要手动设置

## 已知问题

### 已修复
- 修复 Control UI 配置面板显示问题
- 修复 Cerebras 提供商连接问题
- 修复 Matrix E2EE 设置问题

### 待观察
- Google Meet 集成需要进一步优化
- 迁移工具可能需要更多测试

## 升级建议

### 推荐升级
- ✅ 建议所有用户升级到此版本
- ✅ 特别推荐需要 Control UI 的用户升级
- ✅ 特别推荐需要迁移工具的用户升级

### 升级前准备
1. 备份当前配置文件
2. 备份自定义技能和插件
3. 记录当前 Control UI 配置（如有）

### 升级步骤
```bash
# 1. 更新 OpenClaw
openclaw update

# 2. 自动修复配置
openclaw doctor --fix

# 3. 验证 Control UI 配置
openclaw config get controlUI

# 4. 测试 Control UI
openclaw control-ui status
```
