## ⚠️ 需要关注的破坏性变更

### 高优先级 (影响用户体验)
1. **ClawHub优先于npm** (v2026.3.22)
   - 插件安装现在优先从ClawHub获取
   - 影响: 安装命令行为变化

2. **Agent超时时间变更** (v2026.3.22)
   - 默认超时从10分钟增加到48小时
   - 影响: 无配置变化，但行为变化

3. **工具配置默认值变更** (v2026.3.2+)
   - `tools.profile` 默认为 `"messaging"`
   - 需要设置为 `"coding"` 才能使用文件/命令工具

4. **TTS系统大升级** (v2026.4.25)
   - 新增 `/tts latest` 命令
   - 新增多个TTS提供商 (Azure Speech, Xiaomi, Volcengine等)

### 中优先级 (影响配置和兼容性)
1. **环境变量重命名** (v2026.3.22)
   - `CLAWDBOT_*` / `MOLTBOT_*` → `OPENCLAW_*`
   - 需要迁移配置

2. **插件SDK变更** (v2026.3.22)
   - 路径变更为 `openclaw/plugin-sdk/*`
   - 影响: 自定义插件开发者

3. **Matrix插件重写** (v2026.3.22)
   - 需要按照迁移指南操作

4. **Chrome MCP路径移除** (v2026.3.22)
   - 需要运行 `openclaw doctor --fix`

### 低优先级 (开发者/高级用户)
1. **沙箱环境变量注入阻止** (v2026.3.22)
2. **消息发现API变更** (v2026.3.22)
3. **图片生成工具标准化** (v2026.3.22)
4. **语音调用Webhook加固** (v2026.3.22)
